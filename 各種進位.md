## 🔄 進位之間的轉換

### 1. **十進位 ↔ 其他進位**

- **十進位 → 其他進位**
   用「除基取餘法」：不斷除以目標基數，記錄餘數，最後倒過來就是結果。

  - 例：`45₁₀ → 二進位`

    ```
    45 ÷ 2 = 22 ... 1
    22 ÷ 2 = 11 ... 0
    11 ÷ 2 =  5 ... 1
     5 ÷ 2 =  2 ... 1
     2 ÷ 2 =  1 ... 0
     1 ÷ 2 =  0 ... 1
    倒過來：101101₂
    ```

    所以 `45₁₀ = 101101₂`

- **其他進位 → 十進位**
   用「展開法」：每個位數 × (基數^位次)，再加總。

  - 例：`101101₂ → 十進位`

    ```
    1×2^5 + 0×2^4 + 1×2^3 + 1×2^2 + 0×2^1 + 1×2^0
    = 32 + 0 + 8 + 4 + 0 + 1
    = 45
    ```

    所以 `101101₂ = 45₁₀`

------

### 2. **二進位 ↔ 八進位**

- 每 **3 位二進位** 對應 1 位八進位。

  - 例：`101101₂ → ?₈`

    ```
    101101 → (101)(101) → (5)(5)
    = 55₈
    ```

------

### 3. **二進位 ↔ 十六進位**

- 每 **4 位二進位** 對應 1 位十六進位。

  - 例：`101101₂ → ?₁₆`

    ```
    101101 → (0010)(1101) → (2)(D)
    = 2D₁₆
    ```

------

### 4. **八進位 ↔ 十六進位**

- 通常透過 **二進位中轉**：
  - `八進位 → 二進位 → 十六進位`
  - 例：`55₈ = 101101₂ = 2D₁₆`
